# Photo Editor using OpenCV and Streamlit

## Description
This project is a simple photo editing web app built using Streamlit and OpenCV.  
Users can upload an image, apply different editing effects, and download the final edited image.



## Features
- Upload an image  
- Resize image  
- Adjust brightness  
- Adjust contrast  
- Convert to grayscale  
- Apply blur effect  
- Apply warm filter  
- Background blur (portrait effect)  
- Sharpen image  
- Download edited image  

### Extra Features
- Edge detection  
- Sketch effect  
- Cartoon effect  
- Image rotation  



## How the App Works
1. Upload an image  
2. Adjust brightness and contrast using sliders  
3. Apply filters and effects  
4. Preview the edited image  
5. Download the final image  



## Technologies Used
- Python  
- OpenCV  
- Streamlit  
- NumPy  
- PIL  



## How to Run the Project

### Step 1: Clone the repository

### Step 2: Go to the project folder

cd photo-editor-opencv-streamlit


### Step 3: Install required libraries

pip install -r requirements.txt


### Step 4: Run the app

streamlit run app.py
